## gotranslate (forked)


## Usage
* import the gotranslate package
``` go
import (
	"github.com/artificerpi/gotranslate"
)
```
* Example
``` go
gotranslate.QuickTranslation("Hello"), language.English, language.Chinese)
// output: 你好
```
